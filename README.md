# Skad-fitness API

### Installation

```bash
$ npm install
```

### Before run the app

Create new file ```.env``` is project root directory and fill all required vars to run the app.

```bash
PORT - port to run application
DB_URL - database url to connect
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
